# CodVedX – AI/ML Internship Projects

This repository contains all the projects completed during the CodVedX Artificial Intelligence & Machine Learning Internship. The projects cover various domains of AI and ML, including data analysis, predictive modeling, natural language processing, conversational AI, and recommendation systems.

Each project is organized in a separate folder and includes its own source code, documentation, and detailed README file. These projects were developed to strengthen practical skills in Python programming, machine learning workflows, data preprocessing, model development, and software development best practices.

## Projects Included

* Project 1 – Utility Usage Prediction Tool
* Project 2 – Student Performance Prediction System
* Project 3 – AI Based Fake News Detection Tool
* Project 4 – AI Chatbot for Internal Helpdesk
* Project 5 – Smart Recommendation System

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Machine Learning
* Natural Language Processing (NLP)
* Data Visualization

This repository serves as a collection of internship projects demonstrating practical applications of Artificial Intelligence and Machine Learning concepts.
